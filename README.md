ClownMc API is a lightweight plugin that provides an API for ClownMC. It allows you to quickly retrieve server information such as TPS, the online player list, and more, while consuming minimal server resources.

# 📦️ Build

### Requirements

- Java 21 or later
- Git

### Clone the repository

```bash
git clone https://github.com/clownmc/ClownMC-Api.git
cd ClownMC-Api
```

### Linux / macOS

```bash
chmod +x gradlew
./gradlew clean build shadowJar
```

Or simply run:

```bash
./build.sh
```

### Windows

```bat
build.bat
```

or

```bat
gradlew.bat clean build shadowJar
```

### Build Output

After the build completes successfully, the plugin JAR will be located in:

```text
build/libs/
```

The shaded JAR generated by `shadowJar` is the file you should use on your server.


# Config
```yaml
api:
  port: 25566 #This is the port where you want to open the API.
  
  #The options below are endpoints; if "enabled" is set to "true," the corresponding endpoint is active.
  get_tps:
    enabled: true

  get_player_list:
    enabled: true
```

# ☕️ API Endpoints

## Get TPS

Returns the current server TPS.

**Endpoint**

```http
GET /api/v1/tps
```

### Response

```json
{
  "tps": 20.0
}
```

---

## Get Player List

Returns a list of all online players.

**Endpoint**

```http
GET /api/v1/get_player_list
```

### Response

```json
[
  "kenftr",
  "Steve",
  "Alex"
]
```
