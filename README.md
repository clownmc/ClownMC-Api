ClownMc API is a lightweight plugin that provides an API for ClownMC. It allows you to quickly retrieve server information such as TPS, the online player list, and more, while consuming minimal server resources.

# 📦️ Build

### Requirements

- Java 21 or later
- Git

### Clone the repository

```bash
git clone https://github.com/clownmc/ClownMC-Api.git
cd ClownMC-Api
```

### Linux / macOS

```bash
chmod +x gradlew
./gradlew clean build shadowJar
```

Or simply run:

```bash
./build.sh
```

### Windows

```bat
build.bat
```

or

```bat
gradlew.bat clean build shadowJar
```

### Build Output

After the build completes successfully, the plugin JAR will be located in:

```text
build/libs/
```

The shaded (fat) JAR generated by `shadowJar` is the file you should use on your server.
